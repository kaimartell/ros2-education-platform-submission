# Final

Thesis project: a beginner-friendly ROS 2 learning environment with a visual frontend and Dockerized ROS 2 backend.

## Repo layout

- `Pyscript/` — frontend educational UI and visualization
- `ROS2-Workspace/` — ROS 2 Docker workspace and related packages
- `docs/` — architecture, testing, UI, and debugging guidance
- `scripts/` — standard repo commands for local development and smoke tests

## Start here

1. Read `docs/architecture.md`
2. Read `docs/ui-principles.md`
3. Use scripts in `scripts/` instead of inventing ad hoc commands
4. Track active work in `WORKLIST.md`

## Standard workflow

- create or choose one task from `WORKLIST.md`
- define acceptance criteria
- make focused changes
- run relevant smoke tests
- record result and next steps

## Agents

This repo is configured for both:
- Codex via `AGENTS.md`
- Claude Code via `CLAUDE.md`

Both agents should follow the same architecture, UI, and validation guidance.
